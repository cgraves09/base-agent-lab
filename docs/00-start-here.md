# Start here

If you just opened this repo, read this file first.

## What this repo is

A public, in-progress log of one builder learning to ship on [Base](https://docs.base.org/get-started/base). The goal is to leave a usable trail other builders can follow — not just write up the destination, but show the road.

## What this repo is not

- A finished tutorial series. Most folders are stubs that will fill in over time.
- A guide written by an expert. The author is learning in public.
- A speculation playground. Aspirational claims don't get checked in here — only material that's been validated in practice.

## How it's organized

- **`docs/`** — orientation pages and short topic writeups (you're reading one now).
- **`experiments/`** — self-contained, hands-on builds. Each numbered folder is one experiment.
- **`guides/`** — longer how-to material that goes deeper than `docs/`.
- **`prompts/`** — curated teaching prompts for working with AI assistants on Base. These are reusable; they are not a transcript dump.
- **`resources/`** — curated links and references.
- **`NEXT.md`** — what's on deck next.
- **`ROADMAP.md`** — higher-level milestone goals.

## How to use it

If you're new to Base: start with [`01-base-basics.md`](./01-base-basics.md), then skim [`../resources/base-docs.md`](../resources/base-docs.md) for the official sources.

If you're following along: watch [`NEXT.md`](../NEXT.md) and the experiments under [`experiments/`](../experiments/) for the latest.

If you want to run the experiments: each one in `experiments/` will have its own README with setup instructions.
