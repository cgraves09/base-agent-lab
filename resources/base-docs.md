# Base — curated resources

A short, opinionated link list. Goal: give someone walking the same path a clear order to read things in, without drowning them.

## Start here (official)

- **[Get started with Base](https://docs.base.org/get-started/base)** — the canonical entry point.
- **[Base docs home](https://docs.base.org/)** — top-level index for everything else.
- **[Base network information](https://docs.base.org/chain/network-information)** — chain IDs, official RPC URLs, explorer links.

## Tooling references

- **[viem docs](https://viem.sh/)** — the modern TypeScript-first library for talking to EVM chains. Used in every experiment in this repo. Start with the [Public Client](https://viem.sh/docs/clients/public) page.
- **[viem chain metadata](https://github.com/wevm/viem/blob/main/src/chains/definitions/baseSepolia.ts)** — what's actually inside `import { baseSepolia } from "viem/chains"`.

## Explorers

- **[Base Sepolia explorer (Basescan)](https://sepolia.basescan.org/)** — paste any block number, address, or transaction hash to inspect it.
- **[Base mainnet explorer](https://basescan.org/)** — same, for mainnet (real money — only relevant later).
