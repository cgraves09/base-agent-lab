# CLAUDE.md — `base-agent-lab`

This is a **public** repo. It is a learning lab and educational artifact for builders who want to ship on [Base](https://docs.base.org/get-started/base). Everything here is meant to be useful to someone other than the author.

## What lives here

| Folder | Purpose |
|---|---|
| `docs/` | Polished writeups orienting readers to specific topics |
| `experiments/` | Self-contained, hands-on experiments anyone can clone and run |
| `guides/` | Longer-form how-to guides (deeper than `docs/`) |
| `prompts/` | **Curated** teaching prompts only — reusable, generalizable, valuable to other builders. Not a transcript dump. |
| `resources/` | Links, references, curated reading |
| `LEARNED.md` | Distilled, generalizable takeaways — running log of "things worth knowing" |
| `NEXT.md` | Public roadmap of what's being worked on next |
| `ROADMAP.md` | Higher-level milestone goals |

## Curation bar

Anything published here should pass these tests:
- Useful to someone other than me
- Generalizable (not tied to my private context)
- Honest (no fabricated or aspirational results)
- Polished enough to look intentional

## Where the raw working material lives

Working drafts, daily logs, raw prompt archives, half-formed ideas, mistakes, and decision logs live in a **separate private notebook**. When something there proves generalizable, it gets rewritten and promoted into this repo. Material does **not** flow the other direction — nothing private leaks into this public repo.

## How to follow along

This repo is the canonical place to track what's being built. Watch the repo on GitHub or follow the X handle linked in `README.md`.
