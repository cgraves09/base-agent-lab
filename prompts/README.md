# Prompts

This folder is **curated teaching prompts only** — reusable, generalizable, valuable to other Base builders.

It is **not** a transcript dump. It is not every prompt I've ever sent to Claude or any other model. The raw, working, personal archive lives elsewhere where it can stay messy without polluting this one.

When something in that archive proves teaching-worthy, it gets rewritten and promoted here.

## What makes a prompt worth publishing here

A prompt belongs in this folder if it passes all of these:

- **Generalizable** — useful to someone other than me, in a context other than my own. No private context baked in.
- **Tested** — the prompt has actually been run, and the writeup includes what happened.
- **Well-commented** — the prompt is annotated so a reader understands *why* it's structured the way it is, not just what it says.
- **Not a transcript** — it's a teaching artifact, not a snapshot of a conversation.

If a prompt fails any of these, it stays in the private archive until it's reworked.

## Format

When prompts start landing here, they'll follow this shape:

```
# <short title>

**Goal:** what this prompt is for.

**When to use it:** the specific situation it's designed for.

**Prompt:**
> The prompt text, in a code block or quote.

**Notes / annotations:** why each part is there, what to swap in/out, common failure modes.

**Tested with:** model + date.
```
